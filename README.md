# 플러스 파이 인스타그램 링크 페이지

플러스 파이 공식 인스타그램 바이오 링크용 페이지입니다.

## 주요 기능

- **프로필 영역** — 로고, 브랜드명, 소개 문구
- **링크 버튼** — 플러스 파이 홈페이지, 한화생명 홈페이지, 플러스 브랜드 소개
- **유튜브 영상 임베드** — 자동재생 (34초부터 시작)
- **파이 앱 다운로드**
  - 모바일/태블릿: 앱 다운로드 링크 즉시 이동
  - 데스크탑: 스토어 선택 모달 → 앱 다운로드 링크 QR코드 제공
- **공유 모달** — URL 복사 기능
- **반응형** — 모바일에서는 풀스크린, 데스크탑에서는 폰 목업 형태

## 기술 스택

- HTML / CSS / JavaScript (순수 바닐라)
- Pretendard 폰트 (CDN)

## 로컬 실행

```bash
cd linktreecopy
python3 -m http.server 8080
```

브라우저에서 `http://localhost:8080/linktree-sample.html` 접속
